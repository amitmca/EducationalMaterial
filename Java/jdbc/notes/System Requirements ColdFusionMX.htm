AbleCommerce 5.5 CFMX System Requirements

Hardware Requirements:

Minimum and recommended requirements to run AbleCommerce on ColdFusion CFMX Server.

Note:  After initial installation, AbleCommerce is fully administered with a web browser. AbleCommerce supports the most popular web browsers. (latest versions of I.E., Firefox, and Opera)

Processor:

  • Minimum: 1 GHz

  • Recommended: 1.6 GHz or higher

Memory:

  • Minimum: 512 MB (Live websites) and 256 MB (development purposes)

  • Recommended: 1GB or higher

Operating System (Windows or Linux):

Microsoft:

    • Microsoft Windows 2000 (SP3) Server

    • Microsoft Windows 2003 Server

    • Microsoft Windows XP Professional (Evaluation only - not for production use).

 

Linux :

WARNING: AbleCommerce 5.5 CFMX is tested ONLY on the latest releases of Red Hat's Linux Server software.

    • Red Hat Linux 7.2, 7.3, 8.0, 9, AS & ES 2.1, AS & ES 3.0

There are many other Hardware/OS/application server configurations supported by Adobe's ColdFusion CFMX server, we cannot guarantee compatibility with other than recent Red Hat Linux server configurations.  CFMX System Requirements

Disk:

  • Minimum: 50 MB (Plus 400MB for ColdFusion)

  • Recommended: Depends on number and size stores and web pages.

 

Windows 2000/2003 Server Software Requirements:

For all features of AbleCommerce to work properly, the following software must be installed.

  1. Web Server: Microsoft Internet Information Services (IIS) 5.0 or 6.0, or Apache Web Server.  IIS is the default web server pre-installed on Windows server platforms.

  2. Adobe ColdFusion MX 7 : Application server and development language for building fast and powerful Internet applications. AbleCommerce 5.5 supports ColdFusion MX 6.0u3, 6.1, 7.0, 7.0.1*

  3. JDBC Driver: A JDBC driver must be present in your ColdFusion MX environment for the database system you use.  Most of the JDBC drivers come pre-installed with ColdFusion MX server, however in some cases, e.g. MySQL 4.1x, you need to install the JDBC driver separately.  See the section "JDBC Driver Requirements" below for more information.

* Any issues from the latest ColdFusion updates will be posted to the 5.5 CFMX community forums

 

Linux Server Software Requirements:

For all features of AbleCommerce to work properly, the following software must be installed.

  1. Web Server: Apache Web Server is the default web server installed with Red Hat Linux.

  2. Macromedia ColdFusion MX 7 : Application server and development language for building fast and powerful Internet applications. AbleCommerce 5.5 supports ColdFusion MX 6.0u3, 6.1, 7.0, 7.0.1*

  3. JDBC Driver: A JDBC driver must be present in your ColdFusion MX environment for the database system you use.  Most of the JDBC drivers come pre-installed with ColdFusion MX server, however in some cases, e.g. MySQL 4.1x, you need to install the JDBC driver separately.  See the section "JDBC Driver Requirements" below for more information.  

* Any issues from the latest ColdFusion updates will be posted to the 5.5 CFMX community forums

 

Database Requirements:

AbleCommerce ships with support for four database platforms.

  • Microsoft SQL Server 2000 or 2003: http://www.microsoft.com/sql/

  • Microsoft SQL Server 2000 Desktop Engine (MSDE 2000)
    (Free Version of SQL Server): http://www.microsoft.com/sql/msde/

  • MySQL 4.1: http://www.mysql.com/  >> support for MySQL 3.x and 4.0 was discontinued in build 1330 and later.

 

JDBC Connection Requirements:

The pre-installed JDBC drivers that ship with ColdFusion MX server do not cover all the database platforms and their different versions. Some of these platforms and their corresponding JDBC drivers are listed below.

MySQL 4.1

MySQL 4.1 is not supported by default JDBC drivers in CFMX server. At this time you should use Connector/J 3.1.7. The latest version(s) of Connector/J are not backwards compatible and not fully tested.

We will have the latest information posted at http://forums.ablecommerce.com/viewtopic.php?t=1657

  • Download Connector/J 3.1.7: mysql-connector-java-3.1.7.zip

For additional information, please see this ColdFusion technote >> where it says in the technote "mysql-connector-java-3.{n}-bin.jar", just use the "mysql-connector-java-3.1.7-bin.jar" from the .zip download.

PostgreSQL

PostgreSQL is not supported by default JDBC drivers in CFMX server.

  • Download JDBC driver for PostgreSQL : http://jdbc.postgresql.org/download.html

MS-SQL

MS SQLServer is supported by default JDBC drivers in CFMX 7.0 server. However, for performance, licensing or other reasons, one may want to use a different JDBC driver instead of the default driver. Two common alternatives are Microsoft JDBC Driver for SQLServer and jTDS JDBC Driver for SQLServer. Microsoft JDBC driver is not supported by AbleCommerce because it has certain limitations. jTDS is supported.

  • Get jTDS JDBC Driver for Microsoft SQLServer : http://jtds.sourceforge.net/

  • CFMX 6.x server requires DataDirect drivers version 3.3, or higher, with MSSQL Server.
    For more information and a download, see http://www.adobe.com/cfusion/knowledgebase/index.cfm?id=tn_18807

 

Browser Requirements:

To use all the advanced features of the AbleCommerce back-end administration, you will need to be using the latest versions of Microsoft Internet Explorer, Firefox, or Opera, on a Windows system. Other browsers or operating systems may work, but are untested and unsup­ported.

Note: Any standard browser can be used for the retail shopping side of AbleCommerce.  Cookies are not required to be enabled to shop with AbleCommerce.

MysqlPlus Shortcut keys

                                                                      

For the Menus, Shortcut Keys are given below.

Menu Name Shortcut Key
Database Alt + D (To Open Database Menu).
Table Alt + T (To Open Table Menu).
Edit Alt + E (To Open Edit Menu).
Query Alt + Q (To Open Query Menu).
Tools Alt +l (To Open Tools Menu)
Help Alt + H (To Open Help Menu).
